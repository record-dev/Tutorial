### ⚠ 例外発生時 ⚠
エラー： `WSL needs updating` </br>
下記画像のようなエラーが出たら Windows PowerShell を管理者権限で開き、以下のコマンドを入力して実行してアップデートをしてください。
```bash
wsl --update
```
